# What NexTraQ is
NexTraQ is Nexcel’s internal system to track **Leads** and **Outgoing Quotations** from email, assign ownership, enforce follow-ups, and maintain history.

# Roles & permissions
- **Admin / Department Head**: assign owners, approve clients, close leads, manage system.
- **Sales Rep / User**: work assigned leads/quotes, add follow-ups, submit lead completion.

# Daily workflow (Sales Rep)
- Open **Home**
- Clear **Overdue**
- Handle **Due Today**
- Work **My Leads**
- Submit completed leads with outgoing quote reference

# Daily workflow (Admin)
- Assign **Unassigned**
- Review team **Overdue**
- Approve pending clients
- Close leads pending closure
