# Quote lifecycle
A quote is created when an email reaches: `quotes@nexcelservice.net`

# What NexTraQ extracts
- Client name (best effort)
- Quote reference (best effort)
- Quote value: **Total without VAT**
- Currency (if detected)

# What you must verify
- Assign Owner
- Select Verified Client (or request new)
- Set Next Action Due
- Add notes with follow-up dates

# Closing a quote
When outcome is final:
- Mark status: Won / Lost / Cancelled (your configured statuses)
- Add closure notes
- Confirm final value and verified client
