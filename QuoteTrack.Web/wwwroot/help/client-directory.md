# Why verified clients matter
Client linking enables:
- Deduplication
- Pipeline per customer
- Customer history across quotes/leads

# Creating a client
- From Quote Details → **Request New**
- Fill details
- Admin approves
- Client becomes selectable everywhere

# Admin rules
- Avoid duplicates
- Standardize naming (official company name)
- Add key contacts and email when possible
