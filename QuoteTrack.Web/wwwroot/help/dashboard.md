# KPI cards
Use KPIs to quickly see:
- Pipeline value
- Won this month
- Overdue count/value
- Due today count/value
- Lead intake (7 days)

# Triage strip
Use quick filters:
- Overdue
- Due today
- High value
- Unassigned
- Missing follow-up

# Action queue
This is your main list. It is sorted:
Overdue → Due Today → High Value → Newest

# Best practice
No record should remain without:
- Owner
- Next action due
- At least 1 note if it’s being worked

# Dashboard studio
- Drag widgets to rearrange your view
- Layout is saved in your browser
- Theme colors and fonts come from **System Settings**
- Use the extra widgets for status mix, leads vs quotes, owner workload, and 6-month intake trend
