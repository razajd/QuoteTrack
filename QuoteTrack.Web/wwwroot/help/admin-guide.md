# Your daily routine
- Assign unassigned leads
- Review overdue follow-ups for team
- Approve client requests
- Close leads that reps submitted

# What to enforce
- Owner must always be assigned
- Next action due must exist
- Notes must be factual and timestamped
- Verified client should be selected for quotes
