# What is stored
- `.eml`: original email thread (traceability)
- PDF quotations, datasheets
- Any file attached to the incoming email

# How preview should work
- `.eml` renders inside the record page
- PDFs preview in an inline viewer/modal
- Download is always separate

# Best practice
Keep the `.eml` untouched. Use notes for internal actions and follow-ups.
